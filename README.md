# Tone Analyzer — Nano Cortex Edition

A browser-based audio tone analysis tool. No server required.

## Features

- **Microphone** — Real-time FFT analysis from mic input
- **File 1 / File 2** — Load audio files for individual analysis
- **Compare** — Side-by-side FFT comparison between two files

## Usage

Open `index.html` in a browser. No installation needed.
